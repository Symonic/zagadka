This was my first commercial project. I've made it for client who wanted to have a system for puzzle solving, and this is it.
There are puzzles in polish language of diffrent topics from every branch of life.

App has a panel for solving puzzles, but also a admin panel, where owner can modify (CRUD) these puzzles, manage images, manage textes on site.

Technologic stack:
  - Fronted: Vanilla JS, HTML, CSS
  - Backend: Django, Python
  - Database: SQLite3 (Django Build-in)

Project is done and deployed on pythonanywhere.com.
